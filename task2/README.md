# TASK 2:

Необходимо реализовать первое приложение, где запускается http-сервер с graceful shutdown и набором методов:

- **GET /version** – выводит версию API по semver (semantic version – читаем в интернете)
- **POST /decode** – принимает строку json формата `{"inputString":"xxx"}`, где `xxx` - строка в формате base64. На
  выходе необходимо отдать json формата `{"outputString": "yyy"}`, где `yyy` - оригинальная строка, полученная из строки
  формата base64
- **GET /hard-op** – метод, который "спит" от 10 до 20 секунд случайным образом, имитируя долгую обработку данных, и
  отдает случайный 500-ый статус ответа или 200.

Необходимо реализовать второе приложение – **http-клиент**, который запрашивает все методы на нашем сервере по очереди,
а последний запрос отменяет, если его обработка занимает более 15 секунд (постараться сделать таймаут запроса через
context).

В консоль вывести все результаты обработки запросов (для последнего запроса вывести успешность и статус код ответа, если
запрос был выполнен):

v1.0.0

YYY

true, 503
